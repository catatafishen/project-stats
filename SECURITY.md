# Security Policy

## Supported versions

The latest released version on the JetBrains Marketplace is the only supported
version. Please update to the latest release before reporting an issue.

## Reporting a vulnerability

Please report security issues **privately** via GitHub's
[private security advisory](https://github.com/catatafishen/codescape/security/advisories/new)
form. Do not open a public issue for security-sensitive reports.

You should receive a response within 7 days. If the issue is confirmed, a
coordinated disclosure timeline will be agreed before any public release
notes or CVE assignment.
